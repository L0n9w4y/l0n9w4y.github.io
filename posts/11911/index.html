0X03 commit编写

1. 介绍

在 Git 中，每次提交代码，都要写 Commit message（提交说明），否则就不允许提交。这个操作通过 git commit 完成

2. 格式

Commit message 包括三个部分：Header，Body 和 Footer，表示如下：

Header部分只有一行，包括三个字段：type（必需）、scope（可选）和subject（必需）
